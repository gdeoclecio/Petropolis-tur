# Petrópolis Tur 🏔️

## Sobre o projeto

Guia turístico inteligente de Petrópolis (RJ), desenvolvido como projeto pessoal para portfólio. Permite explorar pontos turísticos da cidade com filtro por categoria, busca por nome e página de detalhes de cada local.

## Funcionalidades

- Listagem de pontos turísticos em cards
- Filtro por categoria (Museu, Restaurante, Bar, Cachoeira)
- Busca por nome em tempo real
- Página de detalhes com imagem, descrição e endereço

## Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Java Spring Boot *(em desenvolvimento)*
- MySQL *(em desenvolvimento)*

## Como executar

```bash
# Clone o repositório
git clone https://github.com/gdeoclecio/Petropolis-tur.git

# Entre na pasta
cd Petropolis-tur

# Inicie o servidor local
npx serve .
```
Acesse `http://localhost:3000/pages/index.html` no navegador.
